Cours Javascript - 01_Introduction

Javascript est un langage :

  • Dynamique
  • Principalement côté client
  • Interprêté
  • Orienté objet
  • Javascript n'est pas JAVA !

Javascript est un langage dynamique :

Ce langage va nous permettre de manipuler des contenus html ou des styles CSS et de les modifier en fonction de divers évènements ou variables. Un évènement peut être par exemple un clic d'un utilisateur à un certain endroit de la page tandis q'une variable peut être l'heure de la journée.

Javascript est un langage principalement côté client :

JS s'éxecute côté client, nous allons quand même pouvoir l'utiliser côté serveur à condition que l'on mette en place un environnement favorable, en utilisant, par ex. node.js.

Javascript est un langage interprété :

Pour être interprété JS doit être éxécuté directement dans un logiciel interpréteur : tous les naviguateurs connus possèdent leur interpréteur JS.

Javascript n'est pas JAVA ! :

Certaines personnes ont tendance à confondre les deux langages « Java » et « JavaScript ». Retenez ici que ces deux langages, bien que syntaxiquement assez proches à la base, reposent sur des concepts fondamentaux complètement différents et servent à effectuer des tâches totalement différentes.

On peut mettre du JS directement dans l'élément HTML, dans une balise ouvrante.
