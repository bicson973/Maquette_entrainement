03_Types de données : Les différents types de données

Les 7 différents types de données en JS

  1. String ou chaine de caractères.
  2. Number ou nombre.
  3. Boolean ou booléen.
  4. Null ou vide.
  5. Undefined ou indéfini.
  6. Symbol ou symbole.
  7. Objet ou objet.

Pour connaitre les types de données on utilise la fonction : typeof()

A propos des guillemets "...", des quotes '...' et sur le caractère d'échappement \

En JavaScript, on va pouvoir utiliser indifféremment des guillemets doubles ou des quotes pour entourer une chaîne de caractères et ces deux méthodes vont être strictement équivalentes à la différence d’autres langages comme le PHP par exemple.
Attention :
si votre chaîne contient un caractère qui est le même que le délimiteur de chaîne choisi, il faudra neutraliser ce caractère en l’échappant au moyen d’un antislash ou changer de délimiteur.

1- String

Il faut toujours entourer nos chaines de caracteres (nos textes) avec des apostrophes (des quotes doubles ou simples quotes) ou des guillemets droits ou guillemets doubles.

2- Number

Les variables en JavaScript vont également pouvoir stocker des nombres. Des nombres entiers positifs ou négatifs ou à virgule (décimaux).

3- Boolean

Ce type de données sera utile pour verifier si un test est vrai ou faux

Un booléen, en algèbre est une valeur binaire (soit 0, soit 1). En informatique, le type booléen est un type qui ne contient que deux valeurs : les valeurs true (vrai) et fasle (faux).

4- Null

Null est un type un peu particulier : null absence de valeur non saisie.

5- Undefined

Si on déclare une variable sans lui attribuer de valeur, alors son type sera undefined

6- Symbol & 7- Object

Les objets sont des structures complexes qui vont pouvoir stocker plusieurs valeurs en même temps et que nous étudierons plus tard car il s’agit là d’un sujet complexe à appréhender pour un débutant.

Si on déclare une variable et qu’on lui passe null, alors son type sera Object.
