Cours JavaScript - Variables & constantes

01_Introduction

1- Qu'est-ce qu'une variable ?

Une variable est une boite servant à stocker des informations de manière temporaire comme du texte (une chaîne de caractère), ou un nombre. La variable peut varier et stocker différentes valeurs au fil du temps.

Une variable et une valeur sont deux choses différentes.

2- Règles de déclaration

On dit (comme en php) que l'on "déclare" une variable. On déclare une variable avec les mots-clefs : let ou var

  • Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore : _
  • Le nom d'une variable ne doit contenir que des lettres, des chiffres et des underscore : _
  • Il n'y a pas d'espace dans le nom d'un variable.
  • cf. introduction.js et entrainement.

3- Initialiser une variable

Quand on assigne une valeur à une variable vide on dit que l'on stocke une valeur ou qu'on initialise une variable.
