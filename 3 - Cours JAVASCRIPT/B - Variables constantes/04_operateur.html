04_opérateur : Les opérateurs arithmétiques

Qu'est-ce qu'un opérateur ?

Un opérateur est un symbole qui va effectuer certaines opérations sur les variables et leurs valeurs. Par exemple l'operateur va permettre de multiplier les valeurs de deux variables.

Tableau des opérateurs arithmétiques
opérateurs Nom de l'opération
+ Addition
- Soustraction
* Multiplication
/ Division
% Modulo
** Exponentielle
Modulo
Le reste entier d'une division euclidienne. On divise 5 par 3 le résultat est 1 et il reste 2 : le reste est le modulo
Exponentielle
L’exponentielle correspond à l’élévation à la puissance d’un nombre par un autre nombre. Exemple : 23

Priorité des calculs et utilisation des parenthèses.

Si on écrit let k = 1 - 2 - 3; le résultat est -4 les opérations se font de gauche à droite

A l'exception de l'élévation à la puissance qui sera calculée en partant de la fin cf let m = 2 ** 3 ** 2; qui correspond à 2 3 = 8 puis 8 3 = 512


Nous allons finalement, comme en mathématiques, pouvoir forcer l'ordre de priorité en utilisant des parenthèse pour indiquer qu'une opération doit être effectuée avant les autres cf. let n = (1 - 2) * 3;

Les opérateurs d'affectation

Les opérateurs d'affectation vont nous permettre d'affecter une certaine valeur à une variable ;

Egal "=" est l'opérateur le plus utilisé.

Opérateurs d'affectation
Opérateurs Définition
+= Additionne puis affecte le résultat
-= Soustrait puis affecte le résultat
*= Multiplie puis affecte le résultat
/= Divise puis affecte le résultat
%= Calcul le modulo puis affecte le résultat
