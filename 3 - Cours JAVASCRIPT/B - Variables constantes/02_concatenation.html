02_Concaténation : les litéraux de gabarits

1- La concaténation

Concaténer signifie littéralement "mettre bout à bout". L'opérateur de concaténation est le + :
ATTENTION ! utilisé avec deux chiffres, il sert à aditionner.

2- Concaténation habituelle

Il faut toujours entourer nos chaines de caracteres (nos textes) avec des apostrophes (des quotes doubles ou simples quotes) ou des guillemets droits ou guillemets doubles.

3- Méthode n°3

Il existe une 3ème méthode, introduite plus récemment et qui utilise les accents graves Alt gr + 7èm et le symbol $

exemple : `x contient ${x}`; est l'équivalent de 'x contient ' + x +;
